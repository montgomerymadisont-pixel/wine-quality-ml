
# Wine Quality Prediction (Statistical Learning)

## Overview
This project applies multiple machine learning techniques to predict wine quality based on chemical properties.

## Dataset
- 6,497 wine samples
- 11 chemical features

## Models Used
- Linear Regression
- Ridge Regression
- Lasso Regression
- Subset Selection

## Results
- RMSE: ~0.73
- R²: ~0.29
- Key predictors: alcohol, sulphates, acidity

## Key Insight
Chemical properties alone only partially explain wine quality, suggesting subjective factors play a major role.
